快速开始

第 1 步：安装依赖

cd ink-context-blog
composer install
npm install

第 2 步：配置环境

cp .env.example .env
# 编辑 .env 文件，填入数据库信息

第 3 步：构建资源

npm run build

第 4 步：激活主题

  1. 登录 WordPress 后台
  2. 进入 Appearance > Themes
  3. 激活 Ink Context Blog
